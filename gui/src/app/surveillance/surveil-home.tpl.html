Introduction

Use the tests in this Surveillance area for in the field, real-world verification of your system to the specified 2015 Edition criteria. These tests can be executed on an ad hoc basis, as they do not require any specific data.

C-CDA Validation

The tests in this surveillance area are not linked to any ONC provided test data. Therefore, the (b)(1) Section (ii) "Validate and display" elements are not present. This is due to the fact that the C-CDA Validator on ETT performs strict validation to the gold standard reference files provided by ONC for the 2015 Edition. Outside of the certification testing intent, use the C-CDA validator found at: www.healthit.gov/scorecard. This C-CDA Scorecard uses best practices and quantitative scoring criteria developed by HL7 to improve the implementation of health care standards in C-CDA data.

DirectTrust members can also send a C-CDA for quantitative scoring to the ONC One Click Scorecard: scorecard@direct.hhs.gov

More information and how to use the One Click Score can be found here: https://oncprojectracking.healthit.gov/wiki/display/TechLabTU/ONC+One+Click+Scorecard
